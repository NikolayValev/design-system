# Architecture Decision Records

Use ADRs to document irreversible or high-impact technical choices.

## Naming

- `ADR-0001-<short-title>.md`
- Increment number sequentially.

## When To Write An ADR

- New platform-wide dependency
- Contract or API compatibility decision
- Infra or deployment topology change
- Security model changes

## Required Sections

- Context
- Decision
- Consequences
- Status
